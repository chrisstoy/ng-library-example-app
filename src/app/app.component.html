Example App using a custom Library

This app demonstrates developing an Angular library that is used by mutliple apps. The goal is to allow developers to work on both the library and the consuming application at the same time as seemlessly as possible.

To acheive this, we want to use 'npm link' to publish the library for the app to use. Changes in the library are detected and trigger the App to rebuild and redeploy itself.
